Sử dụng css trong trang html 

            - có 2 cách:
                + style inline: thuộc tính style của tag html chỉ
                áp dụng cho phần tử hiện tại
                + tag style: áp dụng cho toàn bộ trang html 
                    ++ nên viết ở phần head của trang html
                + file .css:(External style)
                    ++ viết 1 lần và áp dụng cho nhiều trang html
                    ++ khi nhúng file css, cũng nên đặt ở phần head

Thuộc tính css

            - margin: khoảng cách giữa các phần tử
                + margin: top / right / bottom / left => margin: 1rem 2rem 3rem 1rem
                + margin: [top, bottom] / [left, right] => margin: 1rem 2rem
                + margin-top: 1rem
                ...
            - padding: khoảng cách giữa nội dung và lề của phần tử
            - boder: đường kẻ viền 
                + border: 1px solid red 
            - border-radius: bo góc
            - width: rộng
            - height: cao
            - position: định vị phần tử (absolute/relative/fixed/...)
            - top/left/right/bottom
            - display: flex / block / ... https://css-tricks.com/snippets/css/a-guide-to-flexbox/
        * màu:
            - màu nền: background-color
            - màu chữ: color 
        * font: 
            - style
            - size
            - color 
            ...    
        * canh lề nội dung:
            text-align: left/right/center/justify
        ...
